## 🚀 Fitur Utama

- ✅ Laporan keuangan otomatis + grafik
- ✅ Pencatatan pengeluaran dengan OCR
- ✅ Klasifikasi kategori otomatis
- ✅ Prediksi pengeluaran bulan depan
- ✅ Goal-based micro saving

---

## 📊 Tech Stack

- **Frontend**: Flutter / React Native
- **Backend**: Python (FastAPI)
- **Data**: Pandas, Jupyter Notebook
- **Database**: PostgreSQL
